Project Description:

This project aims to build a full-stack real-time group chat application using Node.js, Express, MongoDB, HTML, CSS, JavaScript, Socket.io, and SendGrid. The application includes user authentication with email verification using Nodemailer or other suitable packages. It features user login with authentication using Json Web Tokens (JWT) and real-time chat functionality using Socket.io. Users can send and receive messages in real-time, ensuring smooth and instant communication.
